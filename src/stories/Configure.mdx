# AI Assistant App - Component Library

## Overview

Component list for group 4 - AI Assistant App!

## Components

### Chat
- **SendBtn**: Button component for sending messages.

### Footer
- **Footer**: The footer component for the application layout.

### Header
- **ProfileAtom**: Displays the user profile information.
- **TitleAtom**: Shows the title of the page or section.

### Message
- **AiAtom**: Represents messages from the AI in the chat.

### Register Form
- **EmailAtom**: Input field component for the email.
- **PasswordAtom**: Input field component for the password.
- **RegisterBtn**: Button component to submit the registration form.
- **UsernameAtom**: Input field component for the username.

### To-Do
- **DoingAtom**: Represents tasks currently in progress.
- **TaskAtom**: A single task item.
- **TasksToDoAtom**: Displays tasks that need to be done.
- **DoneAtom**: Displays tasks that are completed.

### Login Form
- **EmailLoginAtom**:Input field component for the email login.
- **PasswordLoginAtom**:Input field component for the password login.
- **LoginBtn**:Button component to submit the login form.
- **LoginForm**: The form component that contains all the login input fields and the submit button.
